Памятка "Основные теги HTML"

Чтобы создать простой сайт, необязательно знать все HTML-теги. Для начала можно изучить самые популярные — их достаточно для создания примитивной страницы с текстом, ссылками, картинками, подключёнными стилями и скриптами.

Содержание

  1. Базовые теги
  2. Теги для форматирования текста
  3. Теги для встраивания элементов
  4. Теги для работы с таблицами
  5. Теги для создания форм и кнопок

Базовые теги

Тег Описание
"!DOCTYPE" Показывает браузеру тип документа, сообщает его версию и язык.
"html" Корневой тег, который сообщает браузеру, что это HTML-документ. Все остальные элементы помещаются внутри него.
"head" Контейнер, в который помещаются метаданные документа, не видимые пользователям, но считываемые поисковыми роботами: например, title или style.
"meta" Тег для оформления метаданных документа, используемых браузером для обработки страницы, а поисковиками — для индексации.
"body" Тег, обрамляющий видимую пользователям часть документа. Всё, что вы укажите внутри этого контейнера, отобразится на странице. Тег имеет несколько атрибутов, позволяющих управлять цветами.
"body bgcolor=?" Цвет фона документа в формате RGB.
"body text=?" Цвет текста.
"body link=?" Цвет гиперссылок.
"body vlink=?" Цвет гиперссылок, по которым уже переходили.
"body alink=?" Цвет гиперссылок при нажатии.
"title" Метатег, который задаёт название страницы, отображаемое на вкладке браузера.
"header" Определяет содержимое блока с вводной информацией сайта или группой ссылок.

Теги для форматирования текста

Тег Описание
"h1 - h6" Теги заголовков, от самого большого к самому маленькому.
"b" Жирный текст без придания важности выделенному фрагменту.
"strong" Расставление акцентов в тексте путём выделения его фрагментов полужирным начертанием.
"i" Выделение текста курсивом без придания важности.
"del" Зачёркивает текст, помечая его удалённым.
"s" Отображает перечёркнутый текст.
"ins" Подчёркивает текст, визуально выделяя внесённые изменения.
"u" Подчёркивание без дополнительного акцентирования внимания.
"em" Расставление акцентов путём выделения фрагментов текста курсивом.
"mark" Выделение частей текста жёлтым маркером.
"tt" Имитация текста, набранного на печатной машинке.
"small" Отображение фрагмента с меньшим кеглем шрифта, чем у остального текста.
"sub" Подстрочное начертание символов.
"sup" Надстрочное начертание символов.
"cite" Оформление цитат.
"address" Добавление контактов или подписи автора. При открытии в вею-браузере выделяется курсивом.
"pre" Вывод неформатированного текста с сохранением пробелов и особенностей переносов.
"p" Контейнер для абзаца.
"br" Переносит текст на другую строку без создания абзаца.
"blockquote" Отступы с обеих сторон для оформления цитаты или врезки.
"q" Краткое цитирование.
"dl" Контейнер для размещения термина и его определения.
"dt" Добавление термина.
"dd" Добавление определения понятия.
"dfn" Выделение термина курсивом. Последующий текст должен раскрывать понятие.
"abbr" Указывает, что текст является аббревиатурой или акронимом. Для добавления пояснения используется атрибут title.
"ol" Список с цифрами.
"ul" Список со значками.
"li" Отметка каждого элемента перечня (цифра или значок в зависимости от типа списка).
"a" Добавление гиперссылки в текст. Имеет обязательный атрибут href, в котором указывается ссылка или якорь. Внутри контейнера помещается текст, при нажатии на который происходит переход на другую страницу или другое место на этой же странице.
"code" Выделение фрагмента кода с помощью шрифта monospace.

Теги для встраивания элементов

Тег Описание
"img" Вставка изображения. Имеет атрибут src, который указывает на адрес нужного файла.
"img src="name" align=?" Выравнивание к одной из сторон документа. Например, значение right переместит рисунок в правый край, а left — в левый.
"img src="name" border=?" Позволяет настроить в пикселях толщину рамки вокруг изображения.
"picture" Контейнер для расположения элемента img без source, который даёт браузеру возможность самостоятельно выбрать подходящую картинку.
"audio" Вставка звукового контента.
"video" Вставка видео (поддерживается Ogg, WebM и MP4).
"source" Указывает местоположение файла для video, audio и picture.
"track" Формирует субтитры для video и audio.
"object" Уонтейнер, через который встраиваются мультимедийные файлы. Для передачи параметров применяется тег param.
"hr" Размещает на странице горизонтальную линию. Имеет несколько атрибутов.
"hr size =?" Устанавливает высоту линии.
"hr width=?" Устанавливает ширину линии.
"hr noshade" Убирает тень у линии.
"hr color=?" Задаёт цвет линии.
"script" Определяет выполнение сценария на стороне посетителя сайта. Может содержать сам скрипт или иметь ссылку на внешний источник.
"noscript" Ограничивает фрагмент документа, в котором скрипт не выполняется.

Теги для работы с таблицами

Тег Описание
"table" Размещает таблицу. Все остальные теги для форматирования таблиц должны находиться внутри это контейнера.
"thead" Определяет заголовок.
"tbody" Отмечает тело таблицы.
"td" Создаёт одну ячейку.
"th" Указывает на заголовок ячейки.
"tr" Создание одной строки.
"tfoot" Показывает нижний колонтитул.
"caption" Вставляет подпись. Указывается после тега table.
"col" Позволяет указать ширину и другие параметры одной или нескольких колонок.

Теги для создания форм и кнопок

Тег Описание
"form" Создание форм на странице. С помощью HTML описывается только внешний вид, для выполнения функций требуется запуск соответствующих скриптов на сервере.
"button" Создаёт интерактивную кнопку. Внутри контейнера нужно поместить текст или изображение.
"select multiple name="NAME" size=?" Формирует меню с поддержкой скроллинга.
"select name="NAME"" Создаёт ниспадающее меню.
"option" Описывает каждый отдельный пункт меню.
"input" Формирует поля для добавления пользовательских данных.
"output" Выводит результаты вычислений, сделанных с помощью скрипта.
"label" Оформляет пометку для поля, созданного тегом input.
"textarea" Создаёт большие поля для ввода текста.
